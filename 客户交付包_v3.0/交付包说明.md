# WebRTC语音助手系统 - 交付包说明

## 🎯 系统概述

这是一套完整的WebRTC语音助手系统，集成了：
- **ASR (语音识别)** - 百度语音识别API
- **LLM (大语言模型)** - SiliconFlow GLM-4-9B模型
- **TTS (语音合成)** - 百度语音合成API
- **WebRTC** - 实时音频传输和处理

## 🚀 核心功能

### 1. 实时语音对话
- 支持中文语音输入
- 智能语音检测，过滤背景噪音
- 完整语音捕获，避免分段处理

### 2. 智能对话系统
- 基于GLM-4-9B大语言模型
- 自然语言理解和生成
- 上下文对话记忆

### 3. 高质量语音合成
- 百度TTS API集成
- 自然流畅的语音输出
- 支持多种语音参数调节

## 🔧 技术架构

### 前端 (WebRTC客户端)
- **HTML5 + JavaScript**
- **WebRTC MediaRecorder API**
- **WebSocket实时通信**
- **智能语音检测算法**

### 后端 (Python服务端)
- **asyncio异步框架**
- **WebSocket服务器**
- **多线程音频处理**
- **API集成管理**

### 核心API
- **百度ASR**: 语音识别
- **SiliconFlow**: 大语言模型
- **百度TTS**: 语音合成
